REGRESAR AL PORTAFOLIO

Todos los Formularios en la web, manejan metodos para el envio de las diferentes peticiones que realiza el usuario. Considerando la arquitectura cliente servidor el manejo de las peticiones se hace por medio del modelo http y ahora el modelo https, cada peticion debe de ser atendido en tiempo de respuesta conforme al modelo TCP/IP . Los metodos principales de los formiularios son:

    • Get:

    Este metodo se encarga de enviar la informacion del formulario por medio de la salida estandar (conocida como url), del navegador esto propicia que puedan existir inyecciones de codigo realizando violaciones a la seguridad del sistema

      • Post:

      Este metodo se encarga de enviar la informacion para poder registrar un cambio o un nuevo registro dentro del sistema , enviandola igual por el metodo estandar.


    En resumen los metodos son: ´

    • Get: sirve para enviar y realizar operaciones de lecturas de consulta
      Post: sirve para enviar informacion deun recurso al servidor con el fin de regisrarlo
      Delete:Sirve para eliminar un recurso esecifico dentro del servidor, obteniendo uuna respuesta de actualización
      Put: sirve para almacenar la informacion de un recurso particular y conectarse al servicio para realizar la actualización
      Head: sirve para regresar el codigp de puesta en cualquier cabecera
